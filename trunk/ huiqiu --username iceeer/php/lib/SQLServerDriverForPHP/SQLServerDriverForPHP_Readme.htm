This document contains late-breaking information and other important information that supplements the Microsoft SQL Server 2005 Driver for PHP documentation. You should read this file completely before you install the SQL Server 2005 Driver for PHP.

For the latest SQL Server 2005 Driver for PHP documentation, see the MSDN online documentation.

Installation

When you unpack the downloaded files for this release, you will find the php_sqlsrv_ts.dll and php_sqlsrv.dll files. These files correspond to a thread-safe version and a non–thread-safe version (respectively) of the SQL Server 2005 Driver for PHP. Follow the installation instructions in the "Loading the Driver" topic in the documentation to install the driver.

Note: The php_sqlsrv_ts.dll file is compatible with php5ts.dll, and php_sqlsrv.dll is compatible with php5.dll.

Known Issues

The following is a known issue with the Microsoft SQL Server 2005 Driver for PHP:

  • The syntax for setting configuration options in the php.ini file is incorrect in the product documentation (see the "How to: Configure Error and Warning Handling" and "Logging Activity" topics). The directives that are shown should be prefixed with "sqlsrv.". For example, to turn on logging of connection activity, the following line should be added to the [sqlsrv] section of the php.ini file:

     
    sqlsrv.LogSubsystems = 2
    Note:

    The syntax for setting configuration options with the sqlsrv_configure function is correct in the documentation.

Support

For support questions related to the SQL Server 2005 Driver for PHP, contact Microsoft Help and Support. For more information, see How and when to contact Microsoft Customer Service and Support. Configurations of the following components are supported:

  • Operating systems: All editions (in 32-bit and 64-bit versions) of Windows Server 2008, Windows Vista, Windows Server 2003, and Windows XP.

  • SQL Server: All editions of SQL Server 2008 and SQL Server 2005.

    Note:

    New SQL Server 2008 data types are handled as their down-level equivalent through the SQL Server 2005 ODBC driver.

  • SQL Server Native Client: The SQL Server 2005 ODBC driver (version 9.0.3042).

  • PHP: Versions 5.2.4 through 5.2.6 of the PHP runtime.
